# Sales Performance Dashboard (Power BI)

## Overview
This project is a Power BI dashboard created to analyze sales data from 2020 to 2022.

## Key Insights
- Total Sales: 19M
- Total Profit: 4M
- Total Quantity: 2K
- Transactions: 967

## Features
- Category-wise sales analysis
- Brand-wise performance
- State-wise sales visualization (Map)
- Interactive dashboard with filters

## Tools Used
- Power BI
- Excel
